#### Running Instructions

1. Tomcat Version: Use Apache Tomcat v9.0 for this project.

2. Database Configuration: Update `ApplicationDB.java` (located under src/main/java/com/cs336/pkg) with your db credentials. Modify the strings on lines 13-15 to match your local database settings.

3. Run: Start the server and access the login page.

To quick start testing (and for your convenience), we provided an SQL script `dummydata.sql` to insert sample data into the database. Alternatively, you can manually add data through the website's features as shown in the demo.
