## Instructions

1. Configure your environment and download all dependencies.

    - Can be done by following the ProjectSETUP.pdf on Canvas/Files, or by watching the video in
    the Canvas Modules.

    - Note: This project uses Apache Tomcat **v9.0**. Make sure to use this instead.

2. Load this project into Eclipse.

    - From Eclipse, File -> Import -> Web -> WAR File. Select the FinalProject.war file from this repository.

3. Configure project to use your database. (not needed yet)

    - in `ApplicationDB.java`, line 14-15, change the corresponding Strings to your local database's username and password. 

4. Run the project.

    - Right click project folder -> Run As -> Run on Server

    - Try out the login/logout functionality.
